# Project Overview
This project is focused on embedded systems programming. It aims to provide resources, projects for diverses mcu from (ESP32, Arduino, STM32, TM4C) for developing embedded applications.



# Prerequisites / Toolchain
To work on this project, ensure you have the following tools installed:
- An IDE (e.g., Eclipse, Visual Studio Code)
- C/C++ compiler (GCC recommended)
- Make or CMake for building projects
